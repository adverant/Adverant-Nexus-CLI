# Example Command

This is an example custom command. You can use it by typing `/example` in the chat.

To create your own commands:

1. Create a new `.md` file in this directory
2. Write the prompt or instructions you want Claude to follow
3. Use it by typing `/commandname` (the filename without `.md`)

Example commands you might want to create:
- `/review` - Review code for improvements
- `/test` - Generate tests for the current file
- `/docs` - Generate documentation
- `/refactor` - Suggest refactoring improvements
